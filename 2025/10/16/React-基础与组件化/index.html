React 基础与组件化

\n+## 核心概念

  • 组件：UI 的最小单元，可复用与组合。
  • JSX：声明式语法糖，最终编译为 React.createElement。
  • Props：组件的输入参数，外部不可变。
  • State：组件内部状态，驱动 UI 变化。
    \n+## 函数组件示例 \n+## 受控与非受控组件
  • 受控：状态由 React 管理（如 <input value={v} onChange={...}>）。
  • 非受控：通过 ref 直接读写 DOM 值。
    \n+## 组件组织建议
  • 单一职责与清晰边界；组合胜过继承。
  • 提取通用组件与样式，避免重复实现。
